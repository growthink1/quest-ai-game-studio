# Available Skills (Slash Commands) — Quest AI Learning Game

## Quest-Specific Skills (Pedagogy & Curriculum)
| Command | Purpose |
|---------|---------|
| `/competency-map` | Decompose an AI/ML concept into skill nodes, map prerequisites, generate Neo4j Cypher |
| `/learning-audit` | Audit a module or mechanic against Bloom's, CLT, SDT, and age-band rules |
| `/assessment-review` | Review assessment items for construct validity, distractor quality, Bloom's alignment |
| `/design-system` | Guided section-by-section GDD authoring (Quest 9-section standard with Learning Objective) |
| `/map-systems` | Decompose game concept into systems, map to competency graph, prioritize design order |

## Reviews & Analysis
| Command | Purpose |
|---------|---------|
| `/design-review` | Review a game design document for completeness and consistency |
| `/code-review` | Architectural code review for a file or changeset |
| `/balance-check` | Analyze game balance data and flag outliers |
| `/asset-audit` | Audit assets for naming, sizing, and pipeline compliance |
| `/scope-check` | Analyze feature or sprint scope against original plan, flag scope creep |
| `/perf-profile` | Structured performance profiling with bottleneck identification |
| `/tech-debt` | Scan, track, prioritize, and report on technical debt |
